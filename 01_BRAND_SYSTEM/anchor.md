## Spacing System

### Base Unit
- **8px base-8 system** (Bloomberg/financial terminal standard)
- **Rationale**: Data-dense layouts, precise alignment, institutional feel

### Scale
- 0.5rem (4px) - Micro spacing (badges, tight elements)
- 0.75rem (6px) - Small spacing
- 1rem (8px) - Base unit
- 1.5rem (12px)
- 2rem (16px) - Standard spacing
- 3rem (24px) - Medium spacing
- 4rem (32px)
- 6rem (48px) - Large spacing
- 8rem (64px) - Section padding (desktop)
- 12rem (96px) - Hero padding
- 16rem (128px) - Extra large spacing

### Section Padding
- **Desktop**: 8rem 0 (128px vertical)
- **Tablet**: 6rem 0 (96px vertical)
- **Mobile**: 4rem 0 (64px vertical)

### Container
- **Max Width**: 1440px (wider than standard 1280px for data tables)
- **Padding**: 0 2rem (32px horizontal)
- **Padding Mobile**: 0 1.5rem (24px horizontal)

### Grid Gap
- **Large**: 4rem (64px) - Between major sections
- **Medium**: 2rem (32px) - Between cards, grid items
- **Small**: 1rem (16px) - Between list items, form fields
- **Tiny**: 0.5rem (8px) - Between icons and labels

### Whitespace Philosophy
- **Generous** (1.2-1.5x standard) - More breathing room than typical landing pages
- **Data-Dense Tables**: Tighter spacing (0.8x) for financial data
- **Editorial Content**: Looser spacing (1.5x) for long-form reading

## Aesthetic Standard

### Reference Sites
1. **Bloomberg Terminal** (bloomberg.com) - Data density, financial authority
2. **McKinsey Insights** (mckinsey.com/insights) - Thought leadership, institutional gravitas
3. **Bridgewater Associates** (bridgewater.com) - Understated sophistication, wealth management
4. **Stripe** (stripe.com) - Modern fintech, clean design, developer-first
5. **Linear** (linear.app) - Precision, minimal, high-performance UI

### Visual Language
- **Data-First**: Charts, tables, metrics dominate
- **Institutional**: Corporate, professional, established authority
- **Modern Fintech**: Clean, minimal, high-performance (not stuffy)
- **Editorial Quality**: Long-form content feels like Financial Times, not HubSpot

### Content Vibe
- **Authoritative** (not promotional) - "Here's the data" not "Buy now!"
- **Educational** (not transactional) - "Here's how it works" not "Sign up today!"
- **Analytical** (not emotional) - "12-17% IRR" not "Life-changing returns!"
- **Institutional** (not casual) - University lecture, not sales pitch

### Reading Level
- **16+ (Graduate/Professional)** - MBA, CFA, institutional investor level
- **Complex Sentence Structures**: Multi-clause, subordinate clauses
- **Technical Terminology**: IRR, LTV, NAV, AUM (assume audience knows)
- **Data-Heavy**: Cite sources, reference studies, show methodology

## Banned Elements

### Zero Tolerance (Never Use)
1. **Exclamation points (!)** - True authority doesn't shout
2. **Emoji** - Institutional platforms don't use emoji
3. **Marketing buzzwords**:
   - "Unlock your potential"
   - "Supercharge your wealth"
   - "Game-changing opportunity"
   - "Revolutionary platform"
   - "Transform your portfolio"
   - "Next-level investing"
   - "World-class returns"
   - "Cutting-edge technology"
   - "Innovative solutions"
   - "Seamless experience"
   - "Best-in-class"
   - "Turnkey"
   - "One-stop-shop"

4. **Cheap Design Elements**:
   - Bouncy animations (elastic, spring, bounce)
   - Rainbow gradients
   - Vibrant colors (>40% saturation)
   - Generic stock photos (people smiling at camera)
   - Clip art or illustrations (use data visualizations instead)

5. **Promotional Language**:
   - "Limited time offer"
   - "Act now"
   - "Don't miss out"
   - "Exclusive deal"
   - "Only X spots left"

## Approved Language Patterns

### Instead of Marketing Fluff, Use:

| ❌ Banned | ✅ Approved |
|----------|-----------|
| "Unlock your potential" | "Systematic portfolio analysis against 100,000+ peer benchmarks" |
| "Game-changing opportunity" | "Dubai real estate: 12-17% IRR (historical, 2020-2025)" |
| "Revolutionary platform" | "AI-powered wealth audit engine analyzing $50B+ in portfolios" |
| "Transform your wealth" | "Reallocate 20-25% to international assets (peer average: 45%)" |
| "World-class experts" | "1,000+ certified advisors (average: 15 years experience, 4.7/5 rating)" |
| "Cutting-edge technology" | "Real-time portfolio monitoring with automated risk alerts" |

### Data-Driven Framing
- Always cite sources: "(Source: UBS Global Wealth Report 2026)"
- Always show methodology: "We rate markets using 7 criteria..."
- Always quantify: "8-12% rental yield" not "High returns"
- Always benchmark: "vs. peer average of 55%" not "Great allocation"

## Micro-Interactions & Animations

### Approved (Bloomberg/Linear Style)
- **Easing**: cubic-bezier(0.4, 0.0, 0.2, 1) ONLY
- **Duration**: 0.2s-0.5s (fast, efficient, professional)
- **Effects**:
  - Fade-in (opacity 0 → 1)
  - Slide-up (translateY 10px → 0)
  - Scale subtle (1.00 → 1.02)
  - Brightness adjust (100% → 105%)
- **Hover States**:
  - Buttons: 0.3s transition, slight brightness increase
  - Cards: 0.3s transition, subtle elevation (shadow)
  - Links: Underline fade-in, color shift
- **Scroll Reveals**: IntersectionObserver, fade-up on enter viewport

### Banned (Consumer/Marketing Style)
- ❌ Bouncy animations (elastic, spring)
- ❌ Rotating elements (>180°)
- ❌ Shake/pulse effects
- ❌ Rainbow gradients
- ❌ Infinite loops (except loading states)
- ❌ Aggressive transitions (>0.7s)
- ❌ Large scale changes (>1.05)
